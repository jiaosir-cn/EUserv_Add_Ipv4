# EUserv_Add_Ipv4

一键安装
```
wget -qO- https://cdn.jsdelivr.net/gh/jiaosir-cn/EUserv_Add_Ipv4/ipv6_add_ipv4 .sh|bash
```

手动开启
```
wg-quick up wgcf
```

手动关闭
```
wg-quick down wgcf
```

查看当前状态
```
wg
```
